Preguntas Frecuentes

¿Por que pagar de esta Manera?

Puede ser por varias razones. Entre ellas por las limitaciones que puede tener el vendedor por su lugar de origen o pais de residencia Ejemplo: Venezuela, Nicaragua, Bolivia, entre otros paises que tiene limitaciones para poder aceptar pagos en linea a nivel internacional. Para tener un metodo de pago provicional de rapida integracion. Al poner a prueba un nuevo emprendimiento, el cobro de un producto o servico, etc...

¿Cuanto tiempo dura el proceso de compra y entrega de un producto?

COMPRAR: El proceso de pago con tarjeta de credito o debito desde cualquier. Solo con VISA o MASTERCARD. Suele durar unos 5 minutos ( como Maximo ). La parte de verificacion para enviar el ID de transaccion o (PASO #3 Finalizar) dicen que se procesa de manera instantanea, pero suele tardar unos 5 a 30 minutos en APROBARCE. Despues de terminar el PASO #3 solo debes esperar a la Entrega.

ENTREGA: Depende del vendedor y del Producto a comprar. Pero ya que este sistema de pago esta diseñado para la venta de productos digitales y servicios online. La entrega suele hacerse rapidamente una vez que el vendedor confirma la entrega de los ETH (es decir del pago) unos minutos o como maximo una hora despues el vendedor debe hacer entrega del producto o los detalles para los servicios Online.
